Este script Python converte arquivos PDF em imagens PNG, permitindo personalizar as dimensões das imagens resultantes. Ele é especialmente útil para preparar imagens para apresentações, documentos digitais ou qualquer outro uso que exija um formato de imagem.

FUNCIONALIDADES

- Conversão em lote: Converte múltiplos arquivos PDF de uma vez.
- Personalização de dimensões: Permite definir a largura e altura desejadas para as imagens PNG.
- Criação automática de pasta: Cria a pasta de destino para as imagens PNG se ela não existir.
- Nomenclatura clara: Os arquivos PNG são nomeados de forma intuitiva, indicando o nome original do PDF e o número da página.

USO

pdf_folder: Caminho para a pasta onde estão os arquivos PDF.
png_folder: Caminho para a pasta onde serão salvas as imagens PNG.
pdf_files: Lista com os nomes dos arquivos PDF a serem convertidos.
desired_width e desired_height: Dimensões desejadas para as imagens PNG.
